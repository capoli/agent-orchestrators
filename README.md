# Agent Orchestrators

This repository is a small collection of agent orchestration examples. Each subdirectory is a self-contained example showing a different way to structure multi-agent or multi-tool workflows.

## Examples

- [claude-orchestrator-simple](/Users/oliviercappelle/Projects/ai/agent-orchestrators/claude-orchestrator-simple): a lightweight Claude-based orchestration setup with a small specialist team and simple operating rules
- [claude-saas-orchestrator](/Users/oliviercappelle/Projects/ai/agent-orchestrators/claude-saas-orchestrator): a fuller Claude SaaS delivery system with product, architecture, implementation, QA, security, and DevOps roles
- [claude-codex-gemini-orchestrator](/Users/oliviercappelle/Projects/ai/agent-orchestrators/claude-codex-gemini-orchestrator): a cross-tool workflow where Claude orchestrates, Codex implements, and Gemini provides repo-context and command-driven help

Each example has its own README and local docs for details.
