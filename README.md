# Agent Orchestrators

This repository is a small collection of agent orchestration examples. Each subdirectory is a self-contained example showing a different way to structure multi-agent or multi-tool workflows.

## Examples

- [claude-orchestrator-simple](./claude-orchestrator-simple/README.md): a lightweight Claude-based orchestration setup with a small specialist team and simple operating rules
- [claude-saas-orchestrator](./claude-saas-orchestrator/README.md): a fuller Claude SaaS delivery system with product, architecture, implementation, QA, security, and DevOps roles
- [claude-codex-gemini-orchestrator](./claude-codex-gemini-orchestrator/README.md): a cross-tool workflow where Claude orchestrates, Codex implements, and Gemini provides repo-context and command-driven help

Each example has its own README and local docs for details.
